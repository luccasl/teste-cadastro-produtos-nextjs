
# Next.js Front End

Website em Next.js do teste de cadastro de produtos.


## Setup

Instalar dependências do NPM

```bash
  cd teste-cadastro-produtos-nextjs
  npm i
```

Iniciar serviço do Next.js

```bash
  npm run dev
```

Acessar serviço

```bash
  http://localhost:3000
```
